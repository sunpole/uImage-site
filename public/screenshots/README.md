# Screenshots / Скриншоты

Create one folder per product version, for example `v0.3.0/demo.webp`.

Use optimized WebP, PNG, or JPEG files. Remove private filenames, paths, API keys, customer images, and internal interface details before publishing.

Создавайте отдельную папку для каждой версии продукта, например `v0.3.0/demo.webp`.

Используйте оптимизированные WebP, PNG или JPEG. Перед публикацией удаляйте закрытые имена файлов, пути, ключи API, изображения клиентов и внутренние детали интерфейса.
