# uImage v0.2.29 comparisons

These are static public results of the real v0.2.29 test at 160 pixels. The outline files are black-and-white views of the same vector result. The raster side reuses the approved source images from v0.2.20.

Это статические публичные результаты реального теста v0.2.29 при 160 пикселях. Файлы outline показывают тот же вектор в чёрно-белом виде. Для растровой стороны используются разрешённые исходные изображения из v0.2.20.
