# uImage v0.2.31 comparisons

These are static public results of the real v0.2.31 internal-number experiment at the active 160 px stage. The yacht and mountain vectors use no callouts. The outline files are black-and-white views of the same results. The raster side reuses the owner-approved reduced source images from v0.2.20. The examples show pre-alpha progress and do not claim production readiness or verified real-paint colors.

Это статические публичные результаты реального эксперимента v0.2.31 с внутренними номерами на активном этапе 160 пикселей. Векторы яхты и гор не используют выноски. Файлы outline показывают те же результаты в чёрно-белом виде. Для растровой стороны повторно используются разрешённые владельцем уменьшенные исходники v0.2.20. Примеры показывают прогресс pre-alpha и не обещают производственную готовность или проверенные цвета реальных красок.
